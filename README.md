# Portfolio

Production-ready React portfolio built with Vite.

## Scripts

```powershell
npm.cmd run dev
npm.cmd run build
npm.cmd run preview
npm.cmd run lint
```

## Structure

- `src/App.jsx` composes the page sections.
- `src/assets` contains image assets.
- `src/components` contains reusable UI components.
- `src/data` contains portfolio content used by sections.
- `src/styles` contains global styles.

## Local Development

Install dependencies, then run:

```powershell
npm.cmd run dev
```

Visit the Vite local URL printed in the terminal.
